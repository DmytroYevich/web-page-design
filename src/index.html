This is my rating of programming books I've read
Agenda
  • Head First. Java
  • Herbert Schiltd. Java. The complete reference
  • Cay S. Horstman. Core Java

  1. Head First. Java

    Great book for beginners. The main feature is the way material is explained.
    It's conveyed in a plain spoken manner, as if you are not just a reader but a participant
    of an interesting conversation. The book covers the main aspects of Java Core.

  2. Herbert Schiltd. Java. The complete reference

    An educational guidance written in a scientific manner with detailed explanation. The covered
    material is whole Java Core. This book is recommended to amateurs with specific knowledge of Java.

  3. Cay S. Horstman. Core Java

    A two-volume guidance dedicated to deep learning of Java Core. Scientific language with
    delving into details. A great choice to find out what's happening under the hood.
